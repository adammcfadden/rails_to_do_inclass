##Rails To Do

###Description
This is a simple To Do app built in Rails.

To follow along and build your own version of this app go to <a href="https://www.learnhowtoprogram.com/lessons/to-do-wikipages-wildlife-tracker-monday-classwork" target="#">LearnHowToProgram</a>

###To open the site locally run the following code:

```
git clone https://github.com/adammcfadden/rails_to_do_inclass.git
bundle
rake db:create
rake db:migrate
rails s
```

If testing also run: 

```
rake db:test:prepare
```
